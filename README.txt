== Description ==

<strong>DD Admin Theme</strong> is a perfect <strong>Free WordPress plugin</strong> that convert your WordPress website's admin dashboard into non-WordPess website dashboard. Though WordPress already provide a great dashboard but many of you might bored with classic WordPress dashboard and want some new fancy look then try this plugin.

Plugin setup is very easy. No extra Setting. Activate the plugin and it's converted to <strong>NEW LOOK</strong>.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `dd-admin-theme.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now, your WordPress dashboard has new look.
